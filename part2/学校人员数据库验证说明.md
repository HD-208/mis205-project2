## 🔐 注册验证流程

1. 用户填写注册信息（姓名、邮箱、学号、密码）
2. 系统检查：
   - 邮箱格式是否正确（必须是 @mail.sustech.edu.cn 格式，格式为：学号@mail.sustech.edu.cn）
   - 学号格式是否正确（12开头的8位数字）
   - **姓名、学号、邮箱三者是否与学校人员数据库中的记录完全匹配**
3. 只有验证通过的用户才能成功注册

## 📊 数据库架构

### school_personnel 表
存储预定义的学校人员信息，作为注册验证的依据。

### users 表
系统注册用户表，包含关联到 school_personnel 表的外键。

## 🔗 与 Supabase 集成（完整版本）

在完整的 Supabase 版本中，这个功能可以这样实现：

```sql
-- 在 Supabase 中创建注册验证函数
CREATE OR REPLACE FUNCTION validate_school_personnel(
    p_full_name VARCHAR,
    p_student_employee_id VARCHAR,
    p_email VARCHAR
)
RETURNS BOOLEAN AS $$
BEGIN
    RETURN EXISTS (
        SELECT 1 FROM school_personnel
        WHERE lower(full_name) = lower(p_full_name)
        AND student_employee_id = p_student_employee_id
        AND lower(email) = lower(p_email)
    );
END;
$$ LANGUAGE plpgsql;
```
